# Waving_Simulator
Waving Pro - 2D Frontal Arm Wave Simulator

這是一款專為街舞愛好者與計算機圖形學開發者設計的 Waving（電流舞/波浪舞）全方位動態模擬器。本專案將繁複的街舞肉體控制（如關節隔離、能量傳遞、微步波動）轉化為網頁端的純數學幾何模型，以賽博霓虹（Cyber Neon）火柴人的形式，精準還原能量在人體關節鏈中傳導的視覺錯覺。

由「技術架構師」、「街舞物理顧問」與「產品經理」AI 協作框架共同策劃與開發。

🚀 核心產品特徵

11 節點全身雙手臂連動鏈：全面打破傳統單側手臂模擬，建立由左至右的完整傳導路徑（左指尖 -> 左掌 -> 左腕 -> 左肘 -> 左肩 -> 胸腔 -> 右肩 -> 右肘 -> 右腕 -> 右掌 -> 右指尖）。

高度約束正向運動學（Height-Constrained FK）：採用高度剖面投影演算法，強制鎖定相鄰節點的骨頭長度為恆定常數，100% 杜絕波動時產生「橡膠手臂」的變形崩壞。

非線性停頓感修飾（Sharpness & Isolation）：引入冪函數調校（Power-based Shaping），讓波形在接近頂點時產生微減速與長停留，完美模擬人體韌帶拉扯的「定格隔離感（Pop）」，拒絕平滑無力的海帶感。

身體律動與胸腔平移（Body Groove & Chest Shift）：當波浪通過核心時，胸腔會順應波動方向做側向隔離平移，且膝蓋與軀幹會隨節奏微幅做下壓彈跳（Groove），大幅提升 2D 火柴人的舞蹈生命力。

行動端全面響應式與 Retina 螢幕優化：

自適應視區（Responsive Viewport）：全面支援智慧型手機、平板與桌機。

視網膜高畫質修正：自動讀取裝置的 devicePixelRatio，在高 DPI 螢幕上依然呈現銳利、滿格的霓虹發光線條。

滑動控波（Swipe to Wave）：支援觸控事件，直接在手機螢幕上左右拖拽手指，即可手動且精確地控制波浪停留在任何關節。

側滑抽屜選單：專為手機端設計的玻璃砂效果隱藏控制面板，最大化留出舞蹈舞台空間。

🛠️ 技術棧與底層原理

前端介面：Tailwind CSS (現代暗黑科技感 UI、手機端 Drawer 抽屜規格)

渲染引擎：HTML5 Canvas API (動態增量時間 Delta Time 驅動、動態 Motion Trail 殘影特效)

核心數學方程式：

基礎波動：W_i(t) = sin(omega * t - phi_i)

停頓感修飾：S_i(t) = sign(W_i(t)) * (|W_i(t)|^p)

骨骼恆定約束：dx = sqrt(Length^2 - dy^2)

📦 快速開始與使用說明

本專案之 MVP 採用 單一網頁檔案（Single-file Web App） 架構，無需安裝任何複雜的後端伺服器環境或依賴套件。

儲存檔案：將模擬器的 HTML 原始碼複製，並在電腦中儲存為 waving_simulator.html。

啟動專案：在電腦中雙擊該 HTML 檔案，它會直接在您的瀏覽器（推薦使用 Chrome, Edge, Safari）中執行。

手機把玩：亦可將此檔案部署至免費託管空間（如 GitHub Pages），即可用手機瀏覽器點開體驗絲滑的觸控控波。

🎛️ 面板調校指南

波動速度 (Speed)：調整波浪由下而上或由左至右傳導的頻率。

脈衝寬度 (Pulse Width)：在「單次脈衝模式」下，控制能量波聚集的範圍大小。

停頓感 (Sharpness)：數值越低（如 0.2~0.4），關節在極限角度的停頓（Isolation）越明顯，舞蹈顆粒感越強。

律動幅度 (Body Groove)：控制下半身與膝蓋隨波浪起伏共同彈跳
